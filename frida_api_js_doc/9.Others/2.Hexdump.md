### Hexdump

#### hexdump(target[, options])

从提供的ArrayBuffer或NativePointer目标中生成一个hexdump，还可以选择自定义输出的选项。

比如说:

```js 
const libc = Module.findBaseAddress('libc.so');
console.log(hexdump(libc, {
  offset: 0,
  length: 64,
  header: true,
  ansi: true
}));
```
